Grid is a 2-Dimensional layout system. it can layout entire sites or smaller components. It allows for precise item placement within containers. 
display: grid;
 sets a grid for the container used such as divs, main, sidebar, many others. These contain elements on the page, determines how they are laid out inside the container. ONLY operates on direct child elements. Governed by rows and columns.
