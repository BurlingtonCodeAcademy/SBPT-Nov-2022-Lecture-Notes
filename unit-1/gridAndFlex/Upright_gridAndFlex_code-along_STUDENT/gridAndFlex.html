Grid Heading

Buy Nuts!
Grid is a 2-Dimensional layout system. it can layout entire sites or smaller components. It allows for precise item placement within containers. 
display: grid;
 sets a grid for the container used such as divs, main, sidebar, many others. These contain elements on the page, determines how they are laid out inside the container. ONLY operates on direct child elements. Governed by rows and columns.

Flex Heading

Accomplish

Horizontal or Vertical alignment of elements

Arrangement of elements

Space distribution between & around elements

Quick Notes

It's one-dimensional. Either row or column, but not both at the same time.

Useful for layout on navbars, headers, image-galleries.

Places layout with the container, not each item within it.

Only operate on their children

Sets how a flex item with shrink or grow to fit the space of the container.

Flex Container

Applies to the container and its children

Must be defined in order to be applied.

Justify Content

Simplifies positioning content horizontally.

  1. flex-start
  2. flex-end
  3. center
  4. space-between
  5. space-around
  6. space-evenly

Align Items

Similiar to Justify Content but works along the cross-axis

  1. flex-start
  2. flex-end
  3. center
  4. stretch
  5. baseline

Centering Content

Uses both center from both justify content and align items.

Flex Direction

Default is row.

Can alter to column or even reverse both column and row.

Align Self

Applied to the child element

  1. flex-start
  2. flex-end
  3. center
  4. stretch
  5. baseline

Nesting

A child component can use different flex options for its children
